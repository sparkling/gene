**Business Model: MARKETPLACE-MAKER (Complete)**

Four-Sided Platform:
- Clients ↔ Knowledge (AI interface, reports)
- Clients ↔ Practitioners (one-on-one consulting)
- Clients ↔ Creators (report template marketplace)
- Clients ↔ Clients (peer communities)

**Two-Sided Marketplace Diagram:**
- DEMAND: Individuals, chronic illness patients, biohackers, parents
- SUPPLY: MDs, Functional Medicine, Naturopaths, TCM, Ayurveda, etc.

**Why Marketplace > SaaS:**
| Factor | Pure SaaS | Marketplace |
|--------|-----------|-------------|
| Revenue ceiling | Limited | Unlimited (% of transactions) |
| Network effects | None | Strong |
| Moat | Features (copyable) | Network (hard to replicate) |
| Exit multiple | 5-10x | 10-20x+ |
