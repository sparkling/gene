## TL;DR

ENCODE (Encyclopedia of DNA Elements) provides open-access functional genomics data including 926,535 human candidate cis-regulatory elements (cCREs), ChIP-seq, ATAC-seq, and RNA-seq experiments. The REST API returns JSON with no authentication required. Rate limit: 10 requests/second. Key integration points: `/search/` endpoint for queries, SCREEN for cCRE data, and standardized BED/bigBed file formats.

---

## Overview

The ENCODE Consortium catalogs functional elements in the human and mouse genomes through systematic biochemical assays. ENCODE4 (current phase) focuses on expanding cell type coverage and integrating single-cell technologies.

**Primary Data Types:**
- **cCREs:** 926,535 human candidate cis-regulatory elements
- **ChIP-seq:** Transcription factor and histone modification binding
- **ATAC-seq/DNase-seq:** Chromatin accessibility
- **RNA-seq:** Gene expression (bulk and single-cell)
- **WGBS:** DNA methylation
- **Hi-C/ChIA-PET:** 3D chromatin structure

## API Endpoints

### Base URL

```
https://www.encodeproject.org/
```

### Core Endpoints

| Endpoint | Method | Description |
|----------|--------|-------------|
| `/{accession}/` | GET | Retrieve single object by accession |
| `/search/` | GET | Search across all object types |
| `/report/` | GET | Tabular report view |
| `/batch_download/` | GET | Generate file download URLs |
| `/metadata/` | GET | Download metadata TSV |

### Object Retrieval

Objects are accessed via unique accession numbers:

```bash
# Full path
curl -H "Accept: application/json" "https://www.encodeproject.org/experiments/ENCSR000AKP/"

# Short form (accession only)
curl -H "Accept: application/json" "https://www.encodeproject.org/ENCSR000AKP/"

# With frame parameter
curl -H "Accept: application/json" "https://www.encodeproject.org/ENCSR000AKP/?frame=object"
```

### Search Endpoint

```bash
# Basic search
curl -H "Accept: application/json" "https://www.encodeproject.org/search/?type=Experiment&format=json"

# With filters
curl -H "Accept: application/json" "https://www.encodeproject.org/search/?type=Experiment&assay_title=ChIP-seq&target.label=H3K27ac&format=json"

# Retrieve all results
curl -H "Accept: application/json" "https://www.encodeproject.org/search/?type=Experiment&limit=all&format=json"
```

## Query Parameters

### Frame Parameters

| Parameter | Description | Performance |
|-----------|-------------|-------------|
| (none) | Curated subset of properties | Fastest |
| `frame=object` | Full properties, embedded refs as IDs | Fast |
| `frame=embedded` | Full properties, embedded objects expanded | Slower |

### Search Operators

| Operator | Syntax | Example |
|----------|--------|---------|
| Equality | `field=value` | `assay_title=ChIP-seq` |
| Negation | `field!=value` | `status!=deleted` |
| Less than | `field=lt:N` | `read_length=lt:100` |
| Greater than | `field=gt:N` | `read_count=gt:1000000` |
| Exists | `field=*` | `target=*` |
| Not exists | `field!=*` | `target!=*` |

### Pagination

| Parameter | Default | Description |
|-----------|---------|-------------|
| `limit` | 25 | Results per page (use `all` for complete set) |
| `from` | 0 | Offset for pagination |
| `format` | HTML | Output format (`json` for API) |

## cCRE (Candidate Cis-Regulatory Elements)

### SCREEN Database

**URL:** https://screen.encodeproject.org/

SCREEN provides access to 926,535 human cCREs and 339,815 mouse cCREs classified by biochemical signatures.

### cCRE Classification Types

| Type | Code | Biochemical Signature |
|------|------|----------------------|
| Promoter-like | PLS | High DNase + H3K4me3 + proximal to TSS |
| Proximal enhancer-like | pELS | High DNase + H3K27ac + proximal to gene |
| Distal enhancer-like | dELS | High DNase + H3K27ac + distal from genes |
| DNase-H3K4me3 | DNase-H3K4me3 | High DNase + H3K4me3 + not proximal |
| CTCF-only | CTCF-only | CTCF binding without other marks |

### cCRE Accession Format

```
EH38E[0-9]{7}  # Human GRCh38 (e.g., EH38E1516972)
EH37E[0-9]{7}  # Human hg19
EM10E[0-9]{7}  # Mouse mm10
```

### cCRE BED9+ Format

| Column | Name | Description |
|--------|------|-------------|
| 1 | chrom | Chromosome |
| 2 | chromStart | Start position (0-based) |
| 3 | chromEnd | End position |
| 4 | name | cCRE accession |
| 5 | score | Normalized signal (0-1000) |
| 6 | strand | Always "." |
| 7 | thickStart | Same as chromStart |
| 8 | thickEnd | Same as chromEnd |
| 9 | itemRgb | Classification color |
| 10+ | Extended fields | Classification, signals, etc. |

### cCRE Color Codes

| Classification | RGB | Hex |
|----------------|-----|-----|
| Promoter-like (PLS) | 255,0,0 | #FF0000 |
| Proximal enhancer (pELS) | 255,205,0 | #FFCD00 |
| Distal enhancer (dELS) | 255,205,0 | #FFCD00 |
| DNase-H3K4me3 | 255,195,77 | #FFC34D |
| CTCF-only | 0,176,240 | #00B0F0 |

## Integration Patterns

### Python: Search Experiments

```python
import requests
from typing import Dict, List, Optional

ENCODE_BASE = "https://www.encodeproject.org"

def search_experiments(
    assay_type: str,
    target: Optional[str] = None,
    biosample: Optional[str] = None,
    assembly: str = "GRCh38",
    limit: int = 100
) -> List[Dict]:
    """Search ENCODE experiments."""
    params = {
        "type": "Experiment",
        "assay_title": assay_type,
        "assembly": assembly,
        "status": "released",
        "format": "json",
        "limit": limit
    }

    if target:
        params["target.label"] = target
    if biosample:
        params["biosample_ontology.term_name"] = biosample

    response = requests.get(
        f"{ENCODE_BASE}/search/",
        params=params,
        headers={"Accept": "application/json"}
    )
    response.raise_for_status()

    return response.json().get("@graph", [])

# Example: Find H3K27ac ChIP-seq in K562
experiments = search_experiments("ChIP-seq", target="H3K27ac", biosample="K562")
```

### Python: Download Files

```python
import requests
from pathlib import Path

def download_encode_file(
    accession: str,
    output_dir: Path,
    file_format: Optional[str] = None
) -> Path:
    """Download file from ENCODE."""
    # Get file metadata
    response = requests.get(
        f"{ENCODE_BASE}/files/{accession}/?format=json",
        headers={"Accept": "application/json"}
    )
    response.raise_for_status()
    metadata = response.json()

    # Build download URL
    href = metadata.get("href")
    if not href:
        raise ValueError(f"No download URL for {accession}")

    download_url = f"{ENCODE_BASE}{href}"
    filename = href.split("/")[-1]
    output_path = output_dir / filename

    # Stream download
    with requests.get(download_url, stream=True) as r:
        r.raise_for_status()
        with open(output_path, 'wb') as f:
            for chunk in r.iter_content(chunk_size=8192):
                f.write(chunk)

    return output_path
```

### Python: Parse cCRE BED

```python
from typing import Dict, Generator
import gzip

def parse_ccre_bed(filepath: str) -> Generator[Dict, None, None]:
    """Parse ENCODE cCRE BED9+ file."""
    opener = gzip.open if filepath.endswith('.gz') else open

    with opener(filepath, 'rt') as f:
        for line in f:
            if line.startswith('#') or line.startswith('track'):
                continue

            fields = line.strip().split('\t')
            if len(fields) < 9:
                continue

            yield {
                'chrom': fields[0],
                'start': int(fields[1]),
                'end': int(fields[2]),
                'accession': fields[3],
                'score': int(fields[4]),
                'strand': fields[5],
                'thick_start': int(fields[6]),
                'thick_end': int(fields[7]),
                'item_rgb': fields[8],
                'classification': fields[9] if len(fields) > 9 else None,
                'cell_types': fields[10].split(',') if len(fields) > 10 else []
            }

# Example usage
for ccre in parse_ccre_bed("ENCFF788FYX.bed.gz"):
    if ccre['classification'] == 'PLS':  # Promoter-like
        print(f"{ccre['accession']}: {ccre['chrom']}:{ccre['start']}-{ccre['end']}")
```

  done

### Best Practices

```python
import time
import requests
from functools import wraps

def rate_limited(max_per_second: float = 10):
    """Decorator to rate limit API calls."""
    min_interval = 1.0 / max_per_second
    last_called = [0.0]

    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            elapsed = time.time() - last_called[0]
            if elapsed < min_interval:
                time.sleep(min_interval - elapsed)
            result = func(*args, **kwargs)
            last_called[0] = time.time()
            return result
        return wrapper
    return decorator

@rate_limited(max_per_second=8)  # Stay under 10/s limit
def fetch_encode_object(accession: str) -> Dict:
    """Fetch single ENCODE object with rate limiting."""
    response = requests.get(
        f"https://www.encodeproject.org/{accession}/?format=json",
        headers={"Accept": "application/json"}
    )
    response.raise_for_status()
    return response.json()
```
