# Gut Microbiome Databases

**Parent:** [Microbiome Resources](../_index.md)

## Overview

Gut microbiome databases focus on the microbial communities of the human gastrointestinal tract. These resources catalog species composition, gene content, and associations with health and disease states.

## Data Sources

| Source | Tier | Description |
|--------|------|-------------|
| [GMrepo](./gmrepo/_index.md) | 2 | Curated gut metagenome repository |
| [gutMGene](./gutmgene/_index.md) | 2 | Gut microbiota-gene associations |
| [MetaHIT](./metahit/_index.md) | 1 | Metagenomics of the human gut |

## Integration Notes

MetaHIT provides reference gut metagenome data. GMrepo aggregates curated studies. gutMGene links microbiota to host gene expression. Cross-reference with disease databases for phenotype associations.
