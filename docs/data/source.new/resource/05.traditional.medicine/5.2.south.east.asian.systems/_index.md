# South and East Asian Systems

**Parent:** [Traditional Medicine](../_index.md)

## Overview

This subcategory covers traditional medicine systems from South and East Asia outside of Traditional Chinese Medicine, including Ayurveda (Indian), Kampo (Japanese), and other regional traditions. These resources document medicinal plants, formulations, and therapeutic uses specific to each tradition.

Key resources include IMPPAT (Indian medicinal plants), KampoDB (Kampo medicine), NPACT (natural products for cancer), and TM-MC (Thai traditional medicine).

## Integration Notes

IMPPAT provides comprehensive Ayurvedic plant-compound-target data. KampoDB documents Japanese Kampo formulations with component herbs. Cross-reference with TCM databases for shared herbs and compounds. Link compounds to PubChem for chemical standardization.
