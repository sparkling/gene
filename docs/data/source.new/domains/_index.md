# Health Domains

This directory contains domain-specific views that organize genetic databases by health application areas. These documents aggregate and reference databases from the primary storage locations, providing health-focused perspectives on genetic variants, interventions, and research.

**Important**: Domain documents are "view" documents that reference databases by application area, not primary data storage.

## Domain Document Structure

Each domain document typically includes:

1. **Overview** - Domain scope and key areas
2. **Genetic Databases** - References to relevant databases organized by application
3. **Key Genes & Variants** - Important genetic markers for the domain
4. **Clinical Applications** - How genetic information is used clinically
5. **Research Focus** - Current research priorities and emerging areas
6. **Interventions** - Evidence-based interventions relevant to the domain
7. **References** - Links to primary database documents

## Cross-Domain Considerations

Many genetic variants and interventions span domains. For example:

- **Inflammation** affects cardiovascular, autoimmune, and metabolic domains
- **Methylation** impacts mental health, cancer, and aging
- **Oxidative stress** relates to cardiovascular, neurodegenerative, and cancer domains

Users should consult multiple domain views when researching complex conditions with multiple system involvement.

## Maintenance

Domain views are updated when:
- New databases are added to primary storage
- Research reveals new domain-specific applications
- Clinical guidelines change for domain-specific interventions
- Cross-domain relationships are identified
