### Download Commands

#### List Contents

```bash
aws s3 ls s3://{{BUCKET}}/{{PREFIX}}/ --no-sign-request
```

#### Download Single File

```bash
aws s3 cp s3://{{BUCKET}}/{{PREFIX}}/{{FILE}} ./ --no-sign-request
```

#### Sync Directory

```bash
aws s3 sync s3://{{BUCKET}}/{{PREFIX}}/ ./local/ --no-sign-request
```

#### With Requester Pays

```bash
aws s3 sync s3://{{BUCKET}}/{{PREFIX}}/ ./local/ --request-payer requester
```
