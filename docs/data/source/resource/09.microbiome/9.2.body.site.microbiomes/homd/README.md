# HOMD - Human Oral Microbiome Database

**Category:** [Microbiome](../../../README.md) > [Body Site Microbiomes](../README.md)

## Overview

The Human Oral Microbiome Database (HOMD) provides a comprehensive resource for studying the microbial species present in the human oral cavity. It includes detailed taxonomic information, genome sequences, and phenotypic data for oral bacteria.

HOMD features the Human Oral Microbiome Taxon (HOMT) system, providing standardized taxonomy for cultivated and uncultivated oral species. It includes genome sequences, 16S rRNA sequences, and extensive metadata for each taxon.

HOMD is the authoritative resource for oral microbiome research, dental microbiology, and periodontal disease studies.
