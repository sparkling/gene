# mBodyMap

**Category:** [Microbiome](../../../README.md) > [Body Site Microbiomes](../README.md)

## Overview

mBodyMap is a comprehensive atlas of human microbiome data organized by body site. It integrates microbiome composition data from multiple studies and body locations, enabling comparative analysis across anatomical sites.

The database provides standardized taxonomic profiles, alpha/beta diversity metrics, and visualization tools for comparing microbiomes across body habitats. It covers major body areas including gut, oral, skin, respiratory, and urogenital sites.

mBodyMap is valuable for body-site-specific microbiome research, comparative studies, and understanding microbiome biogeography.
