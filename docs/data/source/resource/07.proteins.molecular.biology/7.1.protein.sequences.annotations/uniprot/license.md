## Summary

UniProt data is available under Creative Commons Attribution 4.0 International License. Free to use, modify, and redistribute for any purpose including commercial use, with attribution.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial
- Redistribute and share
- Modify and create derivative works
- Integrate into products and services
- Bulk download without restriction

---

## Restrictions

What you MUST do:

- Provide attribution to UniProt
- Indicate if modifications were made

---

## Special Terms

### API Access

- No API key required
- Rate limits apply (vary by endpoint)
- Streaming endpoint for large queries

### Bulk Downloads

- Weekly releases available
- Multiple formats: FASTA, XML, RDF, TSV
- FTP: ftp.uniprot.org/pub/databases/uniprot/
