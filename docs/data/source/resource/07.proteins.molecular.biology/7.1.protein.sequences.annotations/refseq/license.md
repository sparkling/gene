# RefSeq License Terms

License details for NCBI RefSeq data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|

---

## Summary

NCBI RefSeq is a product of the US federal government and is in the public domain. There are no restrictions on use, redistribution, or modification. Attribution is customary but not legally required.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial
- Redistribute freely
- Modify and create derivative works
- Integrate into commercial products
- No license required

---

## Restrictions

No legal restrictions. Best practices:

- Cite the source appropriately
- Note that NCBI does not endorse derivative products
- Be aware of third-party sequences with restrictions

---

## Special Terms

### E-utilities API

- Free access with no API key (limited rates)
- API key recommended for higher throughput
- Register at: ncbi.nlm.nih.gov/account/

### Usage Guidelines

- Do not overload NCBI servers
- Off-peak queries recommended for bulk access
- Use FTP for large-scale downloads
