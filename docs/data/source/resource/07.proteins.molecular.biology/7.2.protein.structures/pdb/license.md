License details for Protein Data Bank data usage.

| Property | Value |

---

## Summary

PDB data is in the public domain under CC0 1.0 Universal. No rights reserved. All data is freely available for any use without restriction. Attribution is customary but not required.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial
- Redistribute without restriction
- Modify and create derivative works
- No attribution legally required
- Bulk download freely

---

## Restrictions

No legal restrictions. Best practices:

- Cite original depositors and publications
- Acknowledge PDB/wwPDB
- Do not misrepresent as original work
