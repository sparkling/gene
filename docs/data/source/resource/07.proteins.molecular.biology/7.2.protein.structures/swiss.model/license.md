## Summary

SWISS-MODEL Repository data is available under Creative Commons Attribution-ShareAlike 4.0 International. You may use, share, and adapt the data for any purpose including commercial, provided you give appropriate credit and distribute derivatives under the same license.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial
- Redistribute with attribution
- Modify and create derivative works
- Use in academic and industrial research
- Download bulk data (FTP available)

---

## Restrictions

What you must or cannot do:

- Must provide attribution
- Derivatives must use same license (ShareAlike)
- Must indicate changes if modified
- Must not imply SIB endorsement
- Rate limits apply to API usage

---

## Special Terms

### API Usage

- Rate limit: 1 request/second
- Bulk downloads via FTP preferred
- AutoModel jobs limited per user

### Data Sources

- Models derived from PDB templates
- Quality scores computed by SWISS-MODEL
- Sequences from UniProt

### Server vs Repository

| Component | Access |
|-----------|--------|
| Repository | Free, CC BY-SA 4.0 |
| Modeling Server | Free for academic |
| API | Public with rate limits |
| FTP | Unrestricted download |
