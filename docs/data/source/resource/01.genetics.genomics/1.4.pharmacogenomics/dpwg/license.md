# DPWG License Terms

License details for DPWG (Dutch Pharmacogenetics Working Group) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | KNMP Terms (Dutch); CC BY-SA 4.0 (via PharmGKB) |
| SPDX Identifier | CC-BY-SA-4.0 (PharmGKB access) |
| Commercial Use | Contact KNMP; Yes via PharmGKB |
| Attribution Required | Yes |
| Redistribution | Yes (with attribution) |

---

## Summary

DPWG guidelines have different access terms depending on the source. Access via PharmGKB is under CC BY-SA 4.0, allowing free use with attribution and share-alike requirements. Access via KNMP (Dutch source) requires KNMP membership or specific arrangements. Commercial use of G-Standaard integrated data requires KNMP licensing.

---

## Permissions

What you CAN do with this data:

- Access guidelines via PharmGKB for academic use
- Integrate PharmGKB-accessed content with attribution
- Use for clinical decision support (with appropriate validation)
- Publish analyses with proper citation

---

## Restrictions

What you CANNOT do without additional permission:

- Commercial use of G-Standaard integrated data without KNMP license
- Redistribute without attribution
- Use without share-alike provisions (PharmGKB access)

---

## Special Terms

### Access Pathways

| Source | License | Commercial Use |
|--------|---------|----------------|
| PharmGKB | CC BY-SA 4.0 | Yes (with attribution) |
| KNMP direct | KNMP membership | Contact KNMP |
| G-Standaard | KNMP license | License required |

### PharmGKB CC BY-SA 4.0

When accessed via PharmGKB, DPWG guidelines are under Creative Commons Attribution-ShareAlike, meaning derivative works must be released under the same license.

---

## Attribution Template

When using DPWG guidelines, use this format:

```
DPWG guidelines accessed via PharmGKB (https://www.pharmgkb.org/page/dpwg).
Licensed under CC BY-SA 4.0.
```

### Citation

```
Swen JJ, et al. Pharmacogenetics: From Bench to Byte.
Clin Pharmacol Ther. 2011;89(5):662-673.

For specific guidelines, cite the relevant DPWG publication.
```
