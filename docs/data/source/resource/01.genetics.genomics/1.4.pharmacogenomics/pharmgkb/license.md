## Summary

PharmGKB data is released under the Creative Commons Attribution-ShareAlike 4.0 International (CC BY-SA 4.0) license. This permits commercial use, modification, and redistribution provided appropriate attribution is given and derivative works are released under the same license. Registration is required for bulk downloads but is free.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Integrate into clinical decision support systems
- Redistribute with attribution
- Create derivative databases (under CC BY-SA 4.0)
- Use for pharmacogenomics implementation

---

## Restrictions

What you CANNOT do without additional permission:

- Redistribute without attribution
- Release derivative works under different license
- Remove or alter attribution/license notices

---

## Special Terms

### Registration Requirement

Bulk data downloads require a free PharmGKB account. API access is available after registration.

### Share-Alike Provision

Under CC BY-SA 4.0, if you create a derivative database and distribute it, you must release it under CC BY-SA 4.0 or a compatible license.

### Third-Party Data

Some data in PharmGKB originates from third parties (e.g., FDA labels) and may have additional terms.

## Attribution Template

When using PharmGKB data, use this format:

```
PharmGKB data from https://www.pharmgkb.org/.
Licensed under CC BY-SA 4.0.
```
