# Pharmacogenomics

**Parent:** [Genetics & Genomics](../README.md)

## Overview

Pharmacogenomics databases catalog genetic variants that influence drug response, metabolism, and toxicity. These resources provide clinical guidelines for genotype-guided prescribing and are essential for implementing precision medicine in clinical practice.

Key resources include PharmGKB (comprehensive PGx knowledge), CPIC (clinical dosing guidelines), PharmVar (star allele definitions), and DPWG (Dutch pharmacogenomics guidelines). Together they enable standardized pharmacogenomic implementation.

## Integration Notes

PharmGKB provides comprehensive variant-drug-phenotype annotations. CPIC and DPWG offer clinical guidelines for specific gene-drug pairs. PharmVar standardizes CYP450 and other pharmacogene star allele definitions. Use these together for complete PGx implementation.
