## Summary

PharmVar data is freely available under an open access model. The database provides the official nomenclature for pharmacogene alleles (star alleles) and can be used freely for academic and commercial purposes with appropriate citation. PharmVar is the authoritative source for pharmacogene haplotype definitions used by CPIC, DPWG, and clinical laboratories.

---

## Permissions

What you CAN do with this data:

- Use allele definitions for genotyping assay design
- Integrate into pharmacogenomic testing pipelines
- Use for academic and commercial research
- Redistribute with attribution
- Develop star allele calling software

---

## Restrictions

What you CANNOT do without additional permission:

- Use without proper citation
- Misattribute allele definitions to other sources
- Use PharmVar name/logo inappropriately

---

## Special Terms

### Authoritative Nomenclature

PharmVar maintains the official nomenclature for pharmacogene star alleles. Clinical laboratories should use PharmVar as the reference for allele definitions.

### Allele Submission

Researchers can submit new allele discoveries to PharmVar for official nomenclature assignment.

### Version Tracking

Allele definitions may be updated. Users should track PharmVar version for reproducibility.

---

## Attribution Template

When using PharmVar data, use this format:

```
PharmVar allele definitions from https://www.pharmvar.org/.
```

### Citation

```
Gaedigk A, et al. The Pharmacogene Variation (PharmVar) Consortium:
Incorporation of the Human Cytochrome P450 (CYP) Allele Nomenclature Database.
Clin Pharmacol Ther. 2018;103(3):399-401.

Gaedigk A, et al. PharmVar and the Landscape of Pharmacogenetic Resources.
Clin Pharmacol Ther. 2020;107(1):52-57.
```
