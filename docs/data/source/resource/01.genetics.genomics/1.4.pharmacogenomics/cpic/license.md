# CPIC License Terms

License details for CPIC (Clinical Pharmacogenetics Implementation Consortium) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Open Access |
| SPDX Identifier | Custom (Open Access) |
| Commercial Use | Yes |
| Attribution Required | Yes (per guideline) |
| Redistribution | Yes |

---

## Summary

CPIC guidelines are freely available under an open access model. All guidelines are peer-reviewed and published in open access journals. Users can freely use CPIC guidelines for clinical implementation, education, and research. Citation of the specific guideline publication is required.

---

## Permissions

What you CAN do with this data:

- Use guidelines for clinical decision support
- Integrate into EHR systems
- Use for commercial pharmacogenomics products
- Redistribute guideline content with attribution
- Adapt guidelines for local implementation

---

## Restrictions

What you CANNOT do without additional permission:

- Use without citing the specific guideline
- Misrepresent CPIC recommendations
- Use CPIC name/logo inappropriately

---

## Special Terms

### Citation Requirements

Each CPIC guideline has a specific publication that must be cited. Guidelines are updated periodically, so cite the version used.

### PharmGKB Integration

CPIC data is curated and hosted by PharmGKB. Access via PharmGKB is under CC BY-SA 4.0 terms.

### Clinical Implementation

CPIC guidelines are intended for clinical use. Implementations should:
- Use official CPIC tables and recommendations
- Stay current with guideline updates
- Validate locally before clinical deployment

---

## Attribution Template

When using CPIC guidelines, use this format:

```
CPIC guidelines from https://cpicpgx.org/.
```

### Citation

Cite the specific guideline used. Example format:

```
Crews KR, et al. Clinical Pharmacogenetics Implementation Consortium
Guideline for CYP2D6 and Codeine Therapy: 2021 Update.
Clin Pharmacol Ther. 2021;109(6):1619-1628.
```

General CPIC citation:
```
Relling MV, Klein TE. CPIC: Clinical Pharmacogenetics Implementation Consortium
of the Pharmacogenomics Research Network.
Clin Pharmacol Ther. 2011;89(3):464-467.
```
