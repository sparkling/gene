# ClinVar License Terms

License details for ClinVar data usage.

| Property | Value |

---

## Summary

ClinVar is a public domain resource maintained by NCBI (National Center for Biotechnology Information). As a U.S. government work, ClinVar data is not subject to copyright protection and is freely available for any use without restriction. Attribution is recommended but not legally required.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research without restriction
- Integrate into commercial products and services
- Redistribute in original or modified form
- Publish derived analyses without permission
- Create derivative databases

---

## Restrictions

What you CANNOT do without additional permission:

- None - data is public domain

---

## Special Terms

### NCBI Data Usage

While ClinVar data itself is public domain, NCBI requests users to:
- Not overload NCBI servers with excessive automated requests
- Use E-utilities API responsibly (3 requests/sec without API key, 10 with key)
- Consider downloading bulk files for large-scale analyses

---

## Attribution Template

When attribution is provided (recommended), use this format:

```
ClinVar data from NCBI (https://www.ncbi.nlm.nih.gov/clinvar/).
Public Domain.
```
