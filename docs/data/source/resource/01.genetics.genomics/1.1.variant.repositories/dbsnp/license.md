# dbSNP License Terms

License details for dbSNP data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Public Domain |
| SPDX Identifier | CC0-1.0 |
| Commercial Use | Yes |
| Attribution Required | No (recommended) |
| Redistribution | Yes |

---

## Summary

dbSNP is a public domain resource maintained by NCBI (National Center for Biotechnology Information). As a U.S. government work, dbSNP data is not subject to copyright protection and is freely available for any use without restriction. Attribution is recommended but not legally required.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research without restriction
- Integrate into commercial products and services
- Redistribute in original or modified form
- Publish derived analyses without permission
- Create derivative databases

---

## Restrictions

What you CANNOT do without additional permission:

- None - data is public domain

---

## Special Terms

### NCBI API Usage

While dbSNP data is public domain, NCBI requests users to:
- Limit API requests to 1 request/second (recommended)
- Use bulk downloads for large-scale analyses
- Register for an NCBI API key for higher rate limits

---

## Attribution Template

When attribution is provided (recommended), use this format:

```
dbSNP data from NCBI (https://www.ncbi.nlm.nih.gov/snp/).
Public Domain.
```

### Citation

```
Sherry ST, et al. dbSNP: the NCBI database of genetic variation.
Nucleic Acids Res. 2001;29(1):308-311.
```
