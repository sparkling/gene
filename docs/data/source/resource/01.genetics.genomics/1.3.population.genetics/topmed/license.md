# TOPMed License Terms

License details for TOPMed (Trans-Omics for Precision Medicine) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | dbGaP Data Use Agreement (individual data); Open Access (aggregate) |
| SPDX Identifier | Custom |
| Commercial Use | Varies by study |
| Attribution Required | Yes |
| Redistribution | Restricted (individual); Yes (aggregate) |

---

## Summary

TOPMed has two access tiers: aggregate frequency data (via Bravo browser) is openly accessible, while individual-level genotype and phenotype data requires dbGaP authorization. The TOPMed imputation server is free for academic use. Commercial use terms vary by contributing study.

---

## Permissions

What you CAN do with this data:

- Query Bravo browser for allele frequencies without restriction
- Use the TOPMed imputation server for academic research
- Access individual data after dbGaP approval
- Publish analyses with proper citation

---

## Restrictions

What you CANNOT do without additional permission:

- Access individual-level data without dbGaP authorization
- Redistribute individual-level data
- Commercial use of some study data (check individual study terms)
- Share data beyond approved research use agreement

---

## Special Terms

### Access Tiers

| Data Type | Access | Requirements |
|-----------|--------|--------------|
| Bravo frequencies | Public | None |
| Imputation server | Public | None (academic) |
| Individual genotypes | Controlled | dbGaP approval |
| Phenotypes | Controlled | dbGaP approval + study-specific |

### dbGaP Authorization

Individual-level data access requires:
1. IRB approval
2. Institutional signing official
3. Data Use Agreement
4. NIH eRA Commons account
5. Approved data access request (2-4 weeks)

---

## Attribution Template

When using TOPMed data, use this format:

```
TOPMed data from NHLBI BioData Catalyst.
Aggregate frequencies via Bravo (https://bravo.sph.umich.edu/).
```

### Citation

```
Taliun D, et al. Sequencing of 53,831 diverse genomes from the NHLBI TOPMed Program.
Nature. 2021;590(7845):290-299.
```
