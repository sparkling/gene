## Summary

The 1000 Genomes Project data is openly accessible under principles established by the Fort Lauderdale Agreement for genomic data sharing. Data can be used freely for any purpose including commercial applications, provided proper citation is given. No registration is required for data access.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Use as an imputation reference panel
- Integrate into commercial products
- Redistribute data
- Create derivative datasets

---

## Restrictions

What you CANNOT do without additional permission:

- Claim to be the original source of the data
- Use without proper citation

---

## Special Terms

### Fort Lauderdale Agreement

The 1000 Genomes Project follows community data sharing principles that:
- Encourage rapid public release of genomic data
- Request users respect the contributions of data producers
- Ask for appropriate citation in publications

### Citation Requirements

The project requests that users cite the appropriate publication based on which data release is used.

---

## Attribution Template

When using 1000 Genomes data, use this format:

```
1000 Genomes Project data from IGSR (https://www.internationalgenome.org/).
```

### Citation

```
The 1000 Genomes Project Consortium. A global reference for human genetic variation.
Nature. 2015;526(7571):68-74.

Byrska-Bishop M, et al. High-coverage whole-genome sequencing of the expanded
1000 Genomes Project cohort including 602 trios.
Cell. 2022;185(18):3426-3440.e19.
```
