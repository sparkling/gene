# gnomAD License Terms

License details for gnomAD (Genome Aggregation Database) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Open Data Commons Open Database License (ODbL) 1.0 |
| SPDX Identifier | ODbL-1.0 |
| Commercial Use | Yes |
| Attribution Required | Yes |
| Redistribution | Yes (share-alike) |

---

## Summary

gnomAD data is released under the Open Data Commons Open Database License (ODbL) 1.0, which allows free use for any purpose including commercial applications. Users must provide attribution and release any derived databases under the same license (share-alike provision). This is one of the most permissive licenses among major genomic databases.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Integrate into commercial products with attribution
- Redistribute with attribution and ODbL license
- Create derivative databases (under ODbL)
- Use in clinical applications

---

## Restrictions

What you CANNOT do without additional permission:

- Redistribute without attribution
- Release derivative databases under different license
- Remove or alter attribution notices

---

## Special Terms

### Share-Alike Requirement

Under ODbL, if you create a derivative database and make it publicly available, you must release it under ODbL or compatible license.

### Individual-Level Data

gnomAD releases only aggregate data. Individual-level data from contributing studies may have separate access requirements (e.g., dbGaP for some source studies).

---

## Attribution Template

When using gnomAD data, use this format:

```
gnomAD data from the Genome Aggregation Database (https://gnomad.broadinstitute.org/).
Licensed under ODbL 1.0.
```
