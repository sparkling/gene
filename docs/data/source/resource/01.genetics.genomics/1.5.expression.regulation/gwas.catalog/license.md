## Summary

The NHGRI-EBI GWAS Catalog is released under the Creative Commons Attribution 4.0 International (CC BY 4.0) license. This permits commercial use, modification, and redistribution provided appropriate attribution is given. No registration is required for data access.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Integrate into commercial products
- Redistribute with attribution
- Create derivative databases
- Use for polygenic risk score development

---

## Restrictions

What you CANNOT do without additional permission:

- Use without attribution
- Claim as original work

---

## Special Terms

### EMBL-EBI Terms

Data is hosted by EMBL-EBI and subject to general EMBL-EBI terms of use for web services and APIs.

### Summary Statistics

Full GWAS summary statistics have their own terms from original authors. The catalog metadata (associations, study info) is under CC BY 4.0.

### API Usage

REST API usage should follow reasonable rate limiting. Consider using bulk downloads for large-scale analyses.

## Attribution Template

When using GWAS Catalog data, use this format:

```
GWAS Catalog data from NHGRI-EBI (https://www.ebi.ac.uk/gwas/).
Licensed under CC BY 4.0.
```
