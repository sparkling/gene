## Summary

eQTLGen summary statistics are freely available under an open access model. The data can be used for academic and commercial research with appropriate citation. As a meta-analysis consortium, eQTLGen aggregates data from multiple cohorts, and the summary statistics represent population-level associations.

---

## Permissions

What you CAN do with this data:

- Use for academic research
- Use for commercial applications
- Integrate into GWAS interpretation pipelines
- Use for Mendelian randomization analyses
- Redistribute with attribution

---

## Restrictions

What you CANNOT do without additional permission:

- Use without proper citation
- Access individual-level data (only summary statistics available)

---

## Special Terms

### Summary Statistics Only

eQTLGen provides only summary statistics, not individual-level data. Individual-level data from contributing cohorts may have separate access requirements.

### Genome Build

Data is provided in GRCh37/hg19 coordinates. Users may need to perform liftover for GRCh38 analyses.

### Phase II

Phase II is in progress with single-cell resolution. Check eQTLGen website for updates on data availability.

## Attribution Template

When using eQTLGen data, use this format:

```
eQTLGen data from https://www.eqtlgen.org/.
```

### Citation

```
Vosa U, et al. Large-scale cis- and trans-eQTL analyses identify thousands
of genetic loci and polygenic scores that regulate blood gene expression.
Nat Genet. 2021;53(9):1300-1310.
```
