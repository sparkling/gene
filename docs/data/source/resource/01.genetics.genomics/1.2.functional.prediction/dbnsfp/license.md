# dbNSFP License Terms

License details for dbNSFP data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Academic Free; Commercial Contact Required |
| SPDX Identifier | Custom |
| Commercial Use | Contact authors; some scores excluded |
| Attribution Required | Yes |
| Redistribution | Limited |

---

## Summary

dbNSFP is freely available for academic and non-commercial use. Commercial use requires contacting the authors and may be subject to licensing restrictions. Some integrated prediction scores have their own licenses that may restrict commercial redistribution (e.g., CADD, REVEL). Users should verify license compliance for each predictor used.

---

## Permissions

What you CAN do with this data:

- Use for academic research
- Use for non-commercial clinical research
- Integrate into academic pipelines
- Publish analyses with citation

---

## Restrictions

What you CANNOT do without additional permission:

- Commercial redistribution of the complete database
- Redistribute certain integrated scores commercially (check individual predictor licenses)
- Use without proper citation

---

## Special Terms

### Integrated Score Licenses

dbNSFP integrates 36+ prediction scores. Some have additional restrictions:

| Score | License | Commercial Note |
|-------|---------|-----------------|
| CADD | Academic free; commercial license | Contact CADD |
| REVEL | Academic use | Contact authors |
| MetaSVM/MetaLR | Academic free | Free |
| AlphaMissense | CC BY 4.0 | Permitted |
| SpliceAI | GPLv3/Custom | Contact Illumina |

### Version-Specific Citation

Each dbNSFP version has specific citation requirements. Check the README file included with each release.

## Attribution Template

When using dbNSFP, use this format:

```
dbNSFP data from https://www.dbnsfp.org/.
Citation required per version.
```
