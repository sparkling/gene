# SpliceAI License Terms

License details for SpliceAI data and software usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | GPLv3 (software); Custom (pre-computed data) |
| SPDX Identifier | GPL-3.0-only (software) |
| Commercial Use | Contact Illumina |
| Attribution Required | Yes |
| Redistribution | Yes (software); Limited (data) |

---

## Summary

SpliceAI has different license terms for software and pre-computed data. The SpliceAI software/model is released under GPLv3, allowing open-source use with copyleft requirements. Pre-computed scores from Illumina BaseSpace are freely available for research use, but commercial applications should contact Illumina for licensing.

---

## Permissions

What you CAN do with this data:

- Use pre-computed scores for academic research
- Use the open-source model for non-commercial prediction
- Integrate scores into research pipelines
- Publish analyses using SpliceAI predictions
- Modify and redistribute the software under GPLv3

---

## Restrictions

What you CANNOT do without additional permission:

- Commercial use of pre-computed scores without Illumina license
- Distribute modified software without releasing source code
- Use Illumina trademarks inappropriately

---

## Special Terms

### Software vs Data

| Component | License | Commercial Use |
|-----------|---------|----------------|
| SpliceAI software (GitHub) | GPLv3 | Must release source |
| Pre-computed scores (BaseSpace) | Research use | Contact Illumina |
| Broad Institute Lookup API | Free lookup | Rate limited |

### Illumina Commercial Licensing

For commercial use of pre-computed scores or integration into commercial products, contact Illumina for licensing terms.
