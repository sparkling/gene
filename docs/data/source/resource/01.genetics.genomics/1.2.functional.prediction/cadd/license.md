# CADD License Terms

License details for CADD (Combined Annotation Dependent Depletion) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Academic Free; Commercial License Required |
| SPDX Identifier | Custom |
| Commercial Use | Requires commercial license |
| Attribution Required | Yes |
| Redistribution | Limited |

---

## Summary

CADD scores are freely available for academic and non-commercial research use. Commercial use requires a separate commercial license from the University of Washington. The web scoring service and pre-computed score downloads are available without registration for academic users.

---

## Permissions

What you CAN do with this data:

- Use for academic research without restriction
- Use the web scoring service for small-scale queries
- Download pre-computed scores for academic use
- Integrate into academic pipelines
- Publish analyses with proper citation

---

## Restrictions

What you CANNOT do without additional permission:

- Commercial use without a license
- Redistribute pre-computed scores in commercial products
- Offer CADD scoring as a commercial service

---

## Special Terms

### Commercial Licensing

For commercial use, contact the University of Washington for licensing terms:
- Contact: innovation@uw.edu
- Express Licensing: Available for some applications

### Academic vs Commercial

| Use Case | License Required |
|----------|------------------|
| Academic research | No (free) |
| Clinical research (non-commercial) | No (free) |
| Commercial product integration | Yes |
| Commercial laboratory service | Yes |

## Attribution Template

When using CADD scores, use this format:

```
CADD scores from https://cadd.gs.washington.edu/.
Academic use free; commercial license required.
```
