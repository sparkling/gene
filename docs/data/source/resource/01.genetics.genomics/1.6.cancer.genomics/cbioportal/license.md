# cBioPortal License Terms

License details for cBioPortal for Cancer Genomics data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Open Data Commons Open Database License (ODbL) 1.0 |
| SPDX Identifier | ODbL-1.0 |
| Commercial Use | Yes |
| Attribution Required | Yes |
| Redistribution | Yes (share-alike) |

---

## Summary

cBioPortal data is released under the Open Data Commons Open Database License (ODbL) 1.0, which allows free use for any purpose including commercial applications. Users must provide attribution and release any derivative databases under the same license. The platform aggregates data from multiple cancer genomics studies including TCGA and AACR Project GENIE.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Access via web interface or API without registration
- Integrate into commercial products with attribution
- Download study data in various formats
- Create derivative databases (under ODbL)

---

## Restrictions

What you CANNOT do without additional permission:

- Redistribute without attribution
- Release derivative databases under different license
- Some individual studies may have additional terms

---

## Special Terms

### Study-Specific Terms

While cBioPortal itself is under ODbL, some hosted studies may have additional terms:
- TCGA data is under NIH GDS policy
- AACR Project GENIE has specific data use agreement
- Institutional studies may have restrictions

### API Usage

Public API is available without authentication. Reasonable rate limiting applies for large queries.

### OncoKB Integration

OncoKB annotations displayed in cBioPortal have separate license terms (free for academic; commercial license for commercial use).

## Attribution Template

When using cBioPortal data, use this format:

```
cBioPortal for Cancer Genomics (https://www.cbioportal.org/).
Licensed under ODbL 1.0.
```
