## Summary

BRCA Exchange data is freely available under an open access model as part of the Global Alliance for Genomics and Health (GA4GH) data sharing initiative. The database aggregates BRCA1 and BRCA2 variant classifications from multiple sources and can be used for clinical and research purposes with appropriate citation.

---

## Permissions

What you CAN do with this data:

- Use for academic and clinical research
- Use for commercial applications
- Integrate into diagnostic pipelines
- Redistribute with attribution
- Query via API without restriction

---

## Restrictions

What you CANNOT do without additional permission:

- Use without proper citation
- Misattribute ENIGMA classifications

---

## Special Terms

### Data Sources

BRCA Exchange aggregates data from multiple sources with varying original terms:
- ClinVar (Public Domain)
- LOVD (Open Access)
- ENIGMA consortium (Expert classifications)
- BIC (Historical data)

### ENIGMA Classifications

ENIGMA expert-reviewed classifications are the gold standard. These are curated by the ENIGMA consortium with rigorous evidence review.

### GA4GH Principles

BRCA Exchange follows GA4GH Framework for Responsible Sharing of Genomic and Health-Related Data principles.

### Clinical Use

Data is suitable for clinical variant interpretation. ENIGMA classifications should be prioritized for clinical decisions.

## Attribution Template

When using BRCA Exchange data, use this format:

```
BRCA Exchange data from https://brcaexchange.org/.
Part of the GA4GH data sharing initiative.
```

### Citation

```
Cline MS, et al. BRCA Challenge: BRCA Exchange as a global resource for
variants in BRCA1 and BRCA2.
PLoS Genet. 2018;14(12):e1007752.
```

For ENIGMA classifications:
```
Cite the specific ENIGMA consortium publication for the variant class.
```
