## Summary

OncoKB is free for academic and non-commercial research use. Commercial use requires a license from Memorial Sloan Kettering Cancer Center (MSKCC). OncoKB is an FDA-recognized tumor mutation knowledgebase providing expert-curated clinical interpretation of cancer variants.

---

## Permissions

What you CAN do with this data:

- Use API for academic research (with free registration)
- Query variants for non-commercial clinical research
- Integrate into academic publications
- Use for clinical decision support in non-commercial settings

---

## Restrictions

What you CANNOT do without additional permission:

- Commercial use without license
- Integrate into commercial diagnostic products without license
- Bulk download for commercial redistribution
- Offer as a commercial service

---

## Special Terms

### License Tiers

| User Type | Access | Requirements |
|-----------|--------|--------------|
| Academic | API access | Free registration |
| Non-profit clinical | API access | Free registration |
| Commercial diagnostic | Full access | Commercial license |
| Pharmaceutical | Full access | Commercial license |

### FDA Recognition

OncoKB is FDA-recognized as a public tumor mutation database. This does not affect licensing terms but indicates regulatory acceptance.

### API Token Required

All API access requires a free account and API token. Academic users can register at oncokb.org.

### Rate Limiting

API has rate limits. Contact OncoKB for high-volume academic use cases.
