# Regulatory and Legal Data Sources

**Parent:** [Literature and Knowledge Resources](../README.md)

## Overview

Regulatory databases provide information about clinical trials, drug approvals, and safety data from regulatory agencies. These resources are essential for drug development, pharmacovigilance, and regulatory intelligence.

## Data Sources

| Source | Tier | Description |
|--------|------|-------------|
| [ClinicalTrials.gov](./clinicaltrials.gov/README.md) | 1 | US clinical trials registry |
| [DailyMed](./dailymed/) | 1 | Drug labeling (symlink to pharmaceuticals) |
| [FDA openFDA](./fda.openfda/README.md) | 1 | FDA open data on drugs, devices, foods |

## Integration Notes

ClinicalTrials.gov provides trial registration data. OpenFDA offers API access to adverse events, labeling, and recalls. DailyMed contains official drug labels. Use together for comprehensive regulatory intelligence.
