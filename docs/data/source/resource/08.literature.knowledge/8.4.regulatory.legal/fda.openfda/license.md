# FDA OpenFDA License Terms

License details for FDA OpenFDA data.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Public Domain |
| SPDX Identifier | CC0-1.0 |
| Commercial Use | Yes |
| Attribution Required | No |
| Redistribution | Yes |

---

## Summary

OpenFDA data is public domain as US government work. The harmonized openFDA fields enhance raw data but do not change license terms.

---

## Permissions

What you CAN do:

- Use for any purpose
- Commercial use allowed
- Redistribute data
- Build applications and products

---

## Restrictions

- API rate limits: 240 requests/minute (without key)
- 120,000 requests/day with API key
- Data should not be used for clinical decisions
