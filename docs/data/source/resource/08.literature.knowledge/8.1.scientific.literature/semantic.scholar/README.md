# Semantic Scholar

**Category:** [Literature](../../../_index.md) > [Scientific Literature](../_index.md)

## Overview

Semantic Scholar is an AI-powered research discovery tool developed by the Allen Institute for AI (AI2). It uses machine learning and NLP to analyze papers, extract key information, and surface relevant research through intelligent recommendations.

The platform provides unique features including TLDR (AI-generated paper summaries), citation context analysis, influential citations identification, and research feeds. It covers multiple scientific disciplines beyond biomedicine.

Semantic Scholar is particularly valuable for research discovery, citation analysis, and understanding paper influence and context.
