## Summary

Semantic Scholar data is available under the Open Database License (ODC-BY). You can use, modify, and share the data for any purpose including commercial, provided you give attribution to Semantic Scholar.

---

## Permissions

What you CAN do with this data:

- Use for academic research
- Integrate into commercial products
- Create derivative datasets
- Access via API or bulk downloads

---

## Restrictions

What you CANNOT do without additional permission:

- Remove attribution notices
- Use trademarks improperly
- Exceed API rate limits

---

## Special Terms

### API Access

- Free tier: 100 requests/5 minutes
- Partner tier: Higher limits (apply for access)
- Data dumps available for bulk access

### Datasets

Academic datasets available for research:
- S2ORC (Open Research Corpus)
- Citation graph
- Paper embeddings
