## Summary

OpenAlex data is released under CC0 (public domain dedication). You can use, modify, and redistribute the data for any purpose without restriction. This includes commercial use.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial
- Redistribute entire dataset
- Create derivative works
- Integrate into products and services
- Use API without authentication

---

## Restrictions

What you CANNOT do:

- Nothing - CC0 has no restrictions
