# Europe PMC License Terms

License details for Europe PMC data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | CC0 (metadata) / Various (full text) |
| SPDX Identifier | CC0-1.0 / Various |
| Commercial Use | Yes (metadata) / Varies (full text) |
| Attribution Required | No (metadata) / Varies (full text) |
| Redistribution | Yes (metadata) |

---

## Summary

Europe PMC metadata is available under CC0 (public domain dedication). Full-text articles have individual licenses specified per article. The text-mined annotations are freely available for research use.

---

## Permissions

What you CAN do with this data:

- Use metadata for any purpose
- Access text-mined annotations
- Use APIs without registration (rate-limited)
- Bulk download Open Access content

---

## Restrictions

What you CANNOT do without additional permission:

- Redistribute full-text with restrictive licenses
- Exceed API rate limits
- Claim EMBL-EBI endorsement

---

## Special Terms

### API Access

- No registration required for basic use
- Higher rate limits available with API key
- Contact helpdesk for bulk access needs

### Text Mining

Annotations from SciLite and Europe PMC text mining pipelines are freely available.
