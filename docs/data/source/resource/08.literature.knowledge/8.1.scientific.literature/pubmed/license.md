# PubMed License Terms

License details for PubMed data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|

---

## Summary

PubMed data is produced by the National Library of Medicine (NLM), a US government agency. As a work of the US government, the data is in the public domain and freely available for any use without restriction.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial applications
- Redistribute raw data and derived works
- Integrate into products and services
- Modify and create derivative works

---

## Restrictions

What you CANNOT do without additional permission:

- Claim government endorsement of your product
- Use NLM trademarks improperly
- Misrepresent the source of data

---

## Special Terms

### Publisher Abstracts

While PubMed metadata is public domain, some abstracts may have separate copyright held by publishers. Check individual article rights.

### Rate Limits

API access requires adherence to NCBI usage guidelines:
- 3 requests/second without API key
- 10 requests/second with API key

---

## Attribution Template

When attribution is provided, use this format:

```
PubMed data from the National Library of Medicine (https://pubmed.ncbi.nlm.nih.gov/).
```

### Citation

```
National Library of Medicine. PubMed [Internet]. Bethesda (MD): National Center for
Biotechnology Information; [cited YYYY Mon DD]. Available from: https://pubmed.ncbi.nlm.nih.gov/
```
