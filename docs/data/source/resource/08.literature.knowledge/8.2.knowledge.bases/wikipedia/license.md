## Summary

Wikipedia content is licensed under Creative Commons Attribution-ShareAlike 4.0 (CC BY-SA 4.0). You can use, modify, and distribute the content for any purpose, including commercial, provided you give attribution and share derivatives under the same license.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial
- Copy and redistribute
- Adapt and create derivative works
- Integrate into products

---

## Restrictions

What you CANNOT do:

- Remove attribution
- Use without ShareAlike for derivatives
- Claim content is from other sources

---

## Special Terms

### ShareAlike

Derivatives must be released under CC BY-SA 4.0 or compatible license.

### API Access

- MediaWiki API freely accessible
- Bulk downloads via dumps.wikimedia.org
- Rate limits apply

## Attribution Template

When using Wikipedia content:

```
Content from Wikipedia (https://en.wikipedia.org/wiki/Article_Title).
Licensed under CC BY-SA 4.0 (https://creativecommons.org/licenses/by-sa/4.0/).
```

### Citation

```
Wikipedia contributors. "Article Title." Wikipedia, The Free Encyclopedia.
Wikipedia, The Free Encyclopedia, [date]. Web. [access date].
```
