## Summary

Wikidata content is released under CC0 (public domain dedication). You can use, modify, and redistribute the data for any purpose without restriction, including commercial use. No attribution required.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial
- Redistribute entire dataset
- Create derivative works
- Integrate into any product
- Access via API or SPARQL

---

## Restrictions

What you CANNOT do:

- Nothing - CC0 has no restrictions

---

## Special Terms

### API and SPARQL

- Query Service: query.wikidata.org
- REST API freely accessible
- Bulk dumps available

While not required, attribution is appreciated by the community.
