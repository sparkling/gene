# Knowledge Base Resources

**Parent:** [Literature and Knowledge Resources](../README.md)

## Overview

Knowledge bases provide structured, machine-readable representations of factual knowledge. These resources enable semantic queries, entity linking, and knowledge graph construction for biomedical applications.

## Data Sources

| Source | Tier | Description |
|--------|------|-------------|
| [Wikidata](./wikidata/README.md) | 1 | Collaborative knowledge graph |
| [Wikipedia](./wikipedia/README.md) | 2 | Collaborative encyclopedia |

## Integration Notes

Wikidata provides structured data with extensive cross-references to biomedical databases. Wikipedia offers textual descriptions and context. Both are valuable for entity resolution and knowledge enrichment.
