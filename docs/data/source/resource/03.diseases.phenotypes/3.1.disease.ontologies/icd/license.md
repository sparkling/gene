# ICD License Terms

License details for International Classification of Diseases (ICD) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | CC BY-ND 3.0 IGO (ICD-11) / WHO License (ICD-10) |
| SPDX Identifier | CC-BY-ND-3.0 |
| Commercial Use | Yes (with restrictions) |
| Attribution Required | Yes |
| Redistribution | Yes (no modifications to codes) |

---

## Summary

ICD-11 is distributed under Creative Commons Attribution-NoDerivs 3.0 IGO license, allowing commercial and non-commercial use without modification of the codes. ICD-10 requires a WHO license for commercial use. API access requires registration.

---

## Permissions

What you CAN do with this data:

- Use ICD-11 codes for commercial and non-commercial purposes
- Incorporate into healthcare IT systems and EHRs
- Build clinical decision support tools
- Use for epidemiological surveillance
- Create software using the classification

---

## Restrictions

What you CANNOT do without compliance:

- Modify or adapt ICD codes or their definitions
- Imply WHO endorsement of products
- Use WHO name or emblem without permission
- Redistribute ICD-10 commercially without license

---

## Special Terms

### No Derivatives

The CC BY-ND license prohibits adaptations or modifications to the ICD classification codes themselves.

### ICD-10 Commercial Use

Commercial use of ICD-10 requires contacting WHO for licensing arrangements.
