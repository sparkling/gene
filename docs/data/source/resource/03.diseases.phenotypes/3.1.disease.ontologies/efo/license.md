## Summary

EFO is distributed under the Apache License 2.0, a permissive open source license. This allows unrestricted use, modification, and distribution for both commercial and non-commercial purposes, with attribution required.

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial applications
- Modify and create derivative ontologies
- Redistribute with or without modifications
- Integrate into proprietary systems
- Use for GWAS annotation and trait standardization

## Restrictions

What you CANNOT do without compliance:

- Use EFO trademarks without permission
- Remove attribution notices
- Claim official endorsement
