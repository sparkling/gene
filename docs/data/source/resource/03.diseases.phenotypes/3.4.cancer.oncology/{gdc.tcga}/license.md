# GDC/TCGA License Terms

License details for Genomic Data Commons and The Cancer Genome Atlas data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | NIH Genomic Data Sharing Policy |
| SPDX Identifier | Custom |
| Commercial Use | Yes (open data) |
| Attribution Required | Yes |
| Redistribution | Tiered access |

---

## Summary

GDC/TCGA data is governed by the NIH Genomic Data Sharing (GDS) Policy. Open access data (summary statistics, processed data) is freely available. Controlled access data (individual-level genomic data) requires dbGaP authorization.

---

## Permissions

What you CAN do with this data:

Open Access Data:
- Use for any purpose including commercial applications
- Download without restriction
- Publish analyses with proper attribution

Controlled Access Data (with dbGaP authorization):
- Use for approved research purposes
- Analyze individual-level genomic data

---

## Restrictions

What you CANNOT do without compliance:

- Access controlled data without dbGaP authorization
- Attempt to identify individual research participants
- Redistribute controlled access data
- Use without acknowledging data source

---

## Special Terms

### Open vs. Controlled Access

| Data Type | Access Level |
|-----------|-------------|
| Somatic mutations (MAF) | Open |
| Copy number (gene-level) | Open |
| Gene expression | Open |
| Clinical data | Open |
| Aligned reads (BAM) | Controlled |
| Raw sequences (FASTQ) | Controlled |

### dbGaP Authorization

Controlled access requires applying through dbGaP with institutional support and DAC approval.
