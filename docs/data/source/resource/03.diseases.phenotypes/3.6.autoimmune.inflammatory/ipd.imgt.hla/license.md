# IPD-IMGT/HLA License Terms

License details for IPD-IMGT/HLA immunogenetics database data usage.

| Property | Value |
|----------|-------|

| Commercial Use | Yes (with attribution) |

---

## Summary

IPD-IMGT/HLA applies the Creative Commons Attribution-NoDerivs License to copyrightable parts of the database including sequence alignments. The data can be freely used for commercial purposes with proper attribution. The database asks commercial users to consider funding support.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial applications
- Integrate into HLA typing software
- Build transplant matching systems
- Redistribute with attribution
- Use for pharmacogenomics applications

---

## Restrictions

What you CANNOT do without compliance:

- Create derivative works of the sequence data
- Use without proper citation
- Modify the official allele nomenclature

---

## Special Terms

### Commercial Support

Commercial users are asked to consider funding the database to support its continued maintenance and development.

### WHO Nomenclature

IPD-IMGT/HLA maintains the official WHO HLA nomenclature. Allele names should be used as published.
