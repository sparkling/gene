## Summary

ImmunoBase is an openly accessible database of genetic associations for immune-mediated diseases. The resource provides free access to curated GWAS data, fine-mapping results, and cross-disease analysis for autoimmune conditions.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial applications
- Integrate into autoimmune disease research pipelines
- Build drug target identification tools
- Redistribute association data
- Perform cross-disease genetic analyses

---

## Restrictions

There are no formal restrictions on use of ImmunoBase data. However:

- Primary GWAS data may have separate access requirements
- Source study citations should be included where appropriate
- Data currency should be verified for clinical applications
