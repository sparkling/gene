## Summary

Orphanet data (via Orphadata) is distributed under the Creative Commons Attribution 4.0 International License. This permissive license allows unrestricted use including commercial applications, with proper attribution to Orphanet required.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial applications
- Integrate into rare disease diagnostic tools
- Create derivative databases and analyses
- Redistribute with attribution
- Build orphan drug development tools

---

## Restrictions

What you CANNOT do without compliance:

- Use without proper attribution to Orphanet
- Claim official endorsement by Orphanet/INSERM
