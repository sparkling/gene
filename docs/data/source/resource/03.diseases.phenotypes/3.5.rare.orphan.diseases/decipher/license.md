# DECIPHER License Terms

License details for DECIPHER genomic variation database data usage.

---

| Property | Value |
|----------|-------|

| SPDX Identifier | Custom |

---

## Summary

DECIPHER uses a tiered data access model. Aggregate summary data and gene-level information is openly available. Patient-level data requires a Data Sharing Agreement with the DECIPHER consortium. Commercial entities can display data via Data Display Agreement.

---

## Permissions

What you CAN do with this data:

Open Data:
- Access aggregate variant statistics
- View gene-level pathogenicity scores
- Use for research and clinical interpretation
- Cite in publications

With Data Sharing Agreement:
- Access patient-level variant and phenotype data
- Contact other depositing centers for patient matching

---

## Restrictions

What you CANNOT do without additional permission:

- Access individual patient records without agreement
- Redistribute patient-level data
- Identify individual patients
- Commercial use of patient-level data without arrangement

---

## Special Terms

### Data Display Agreement

Third-party variant analysis companies and genome browsers (e.g., Ensembl, UCSC) can display DECIPHER data under a Data Display Agreement.

### Clinical Center Registration

Depositing centers must register and agree to data sharing terms.
