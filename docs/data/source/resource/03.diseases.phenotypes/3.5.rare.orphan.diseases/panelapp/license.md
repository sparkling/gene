---

## Summary

PanelApp data is openly accessible without registration or licensing requirements. The platform is designed for broad use in clinical genomics, and gene panel data can be freely accessed via the web interface and REST API.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial applications
- Integrate into clinical diagnostic pipelines
- Build variant interpretation tools
- Redistribute gene panel data
- Create derivative panel designs

---

## Restrictions

There are no formal restrictions on use of PanelApp data. However:

- Evidence levels should be appropriately communicated
- Panels evolve over time; verify currency for clinical use
- Clinical decisions should consider local context
