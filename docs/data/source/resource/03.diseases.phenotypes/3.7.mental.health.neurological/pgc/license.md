# PGC License Terms

License details for Psychiatric Genomics Consortium (PGC) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | PGC Data Use Agreement |
| SPDX Identifier | Custom |
| Commercial Use | Requires permission |
| Attribution Required | Yes |
| Redistribution | Tiered access |

---

## Summary

PGC data access is tiered. GWAS summary statistics are freely available for academic use. Individual-level genotype data requires dbGaP controlled access authorization. Commercial use requires advance permission from the PGC Data Access Committee.

---

## Permissions

What you CAN do with this data:

Summary Statistics (Open):
- Use for academic research
- Perform polygenic risk score analyses
- Calculate genetic correlations
- Publish with proper citation

Individual-Level Data (with dbGaP approval):
- Approved research analyses
- Meta-analyses with PGC collaboration

---

## Restrictions

What you CANNOT do without additional permission:

- Commercial or for-profit use without DAC permission
- Develop predictive tests for unborn individuals
- Access individual data without dbGaP authorization
- Redistribute individual-level data

---

## Special Terms

### Research Use Only

Summary statistics are for scientific research only. Commercial use requires advance permission.

### Predictive Tests

Any predictive test use must acknowledge experimental nature and non-genetic etiological factors.

---

## Attribution Template

When using PGC data:

```
Summary statistics were obtained from the Psychiatric Genomics Consortium
(https://pgc.unc.edu/). Cite the relevant study publication.
```

### Citation

Cite the specific PGC study (e.g., PGC-SCZ3, PGC-MDD3) when using data.
