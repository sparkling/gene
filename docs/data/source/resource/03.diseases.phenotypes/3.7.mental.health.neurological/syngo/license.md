## Summary

SynGO is distributed under the Creative Commons Attribution 4.0 International License (CC BY 4.0). This permissive license allows unrestricted use including commercial applications, with proper attribution and citation required.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial applications
- Integrate into neurological disease research pipelines
- Build synaptic gene enrichment tools
- Redistribute with attribution
- Create derivative analyses

---

## Restrictions

What you CANNOT do without compliance:

- Use without proper attribution
- Fail to cite the SynGO publication

---

## Attribution Template

When attribution is required, use this format:

```
SynGO data from the SynGO Consortium (https://syngoportal.org/).
License: CC BY 4.0.
```

### Citation

```
Koopmans F, et al. (2019) "SynGO: An Evidence-Based, Expert-Curated Knowledge Base
for the Synapse." Neuron. 103(2):217-234.e4.
```
