## Summary

Open Targets Platform data is distributed under the Creative Commons Attribution 4.0 International License (CC BY 4.0). This permissive license allows unrestricted use including commercial applications, with proper attribution required.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial applications
- Integrate into drug discovery pipelines
- Create derivative databases and analyses
- Redistribute with attribution
- Build target validation tools

---

## Restrictions

What you CANNOT do without compliance:

- Use without proper attribution
- Claim endorsement by Open Targets partners

---

## Attribution Template
