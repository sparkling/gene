License details for eBASIS bioactive compound data usage.

---

## Summary

eBASIS data is managed by EuroFIR AISBL. Academic and research use is available through free registration and data use agreement. Commercial applications require a separate license negotiation with EuroFIR.

---

## Permissions

What you CAN do with this data:

- Access for academic research with registration
- Use data in non-commercial publications
- Reference and cite in academic works
- Query via web interface

---

## Restrictions

What you CANNOT do:

- Redistribute data without permission
- Use for commercial purposes without license
- Bulk download without agreement
- Integrate into commercial products

---

## Special Terms

### Academic Access

1. Register at ebasis.eurofir.org
2. Accept data use agreement
3. Access for non-commercial research

### Commercial Licensing

Contact EuroFIR for commercial use terms:
- Data integration licenses
- API access
- Bulk data access
