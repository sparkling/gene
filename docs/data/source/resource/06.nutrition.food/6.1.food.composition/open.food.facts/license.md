# Open Food Facts License Terms

License details for Open Food Facts data usage.

| Property | Value |

---

## Summary

Open Food Facts data is available under the Open Database License (ODbL) 1.0. You are free to use, modify, and share the data for any purpose, including commercial use, provided you attribute the source and share any derivative databases under the same license.

---

## Permissions

What you CAN do with this data:

- Use for any purpose, including commercial applications
- Create derived databases and analyses
- Redistribute the data with attribution
- Build commercial products using the data
- Modify and enhance the database

---

## Restrictions

What you MUST do:

- Provide attribution to Open Food Facts
- Keep any derived databases under ODbL (share-alike)
- Indicate if you have modified the data

---

## Special Terms

### Image License

Product images are licensed under Creative Commons Attribution-ShareAlike 3.0 (CC BY-SA 3.0), which requires attribution and share-alike for any image use.

### API Usage

- No API key required for public access
- Rate limit: ~100 requests/minute recommended
- Bulk operations: Use download files instead of API

---

## Attribution Template

When attribution is required, use this format:

```
Data from Open Food Facts (https://world.openfoodfacts.org/).
License: Open Database License (ODbL).
```

### Citation

```
Open Food Facts contributors.
Open Food Facts - World's largest open food products database.
https://world.openfoodfacts.org/
License: ODbL 1.0
```
