## Summary

Exposome-Explorer data is available under Creative Commons Attribution 4.0 International License. The database is freely accessible for any purpose, including commercial use, with proper attribution. Maintained by IARC/WHO.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial
- Redistribute and share
- Adapt and build upon
- Create derivative works
- Integrate into products and services

---

## Restrictions

What you MUST do:

- Provide attribution to Exposome-Explorer and IARC
- Indicate if modifications were made
- Not imply IARC/WHO endorsement

## Special Terms

### No API

- No formal REST API available
- Web interface for browsing and search
- TSV downloads available for bulk access

### Academic Use

- Cite the main publication
- Reference database URL
