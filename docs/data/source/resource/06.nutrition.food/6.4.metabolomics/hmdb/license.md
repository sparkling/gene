## Summary

HMDB data is available under Creative Commons Attribution-NonCommercial 4.0 International License. Academic and non-commercial use is freely permitted with attribution. Commercial applications require a separate licensing agreement.

---

## Permissions

What you CAN do with this data:

- Use for academic research and publications
- Integrate into non-commercial tools
- Redistribute with attribution (non-commercial)
- Create derivative works for research
- Download bulk data for research

---

## Restrictions

What you CANNOT do without permission:

- Use in commercial products or services
- Sell access to HMDB data
- Use in revenue-generating applications
- Redistribute commercially

## Special Terms

### API Access

- REST API freely available
- Format options: JSON, XML
- Bulk downloads: hmdb.ca/downloads
- No API key required

### Commercial Licensing

Contact Wishart Research Group for commercial use:
