# Natural Medicines Comprehensive Database

**Category:** [Nutrition](../../../README.md) > [Dietary Supplements](../README.md)

## Overview

The Natural Medicines Comprehensive Database (formerly Natural Medicines by TRC Healthcare) is a clinical decision support resource providing evidence-based information about natural products, dietary supplements, and complementary therapies.

The database includes detailed monographs with effectiveness ratings, safety grades, drug interactions, dosing guidelines, and mechanism of action information. Each entry is graded using a systematic methodology based on available clinical evidence.

Natural Medicines is widely used by healthcare professionals, researchers, and pharmacists for evidence-based decision making about natural products. It requires subscription access.
