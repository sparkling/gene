## Summary

DSLD is a US government database maintained by the NIH Office of Dietary Supplements. As a federal resource, the data is in the public domain and free for any use. API access requires free registration to obtain an API key.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial
- Redistribute freely
- Create derivative works
- Integrate into commercial products
- No attribution legally required (but recommended)

---

## Restrictions

What you MUST do:

- Register for free API key for programmatic access
- Follow API rate limits
- Not misrepresent data as official endorsement

---

## Special Terms

### API Registration

- Free registration at: https://api.ods.od.nih.gov/signup
- API key sent via email
- Include key in X-Api-Key header

### Rate Limits

- No strict limit published
- Reasonable use expected
- Bulk download preferred for large data sets
