# DrugBank License Terms

License details for DrugBank database data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | CC BY-NC 4.0 (Academic) |
| SPDX Identifier | CC-BY-NC-4.0 |
| Commercial Use | Subscription required |
| Attribution Required | Yes |
| Redistribution | Non-commercial only |

---

## Summary

DrugBank offers tiered licensing. Academic users can access the database under CC BY-NC 4.0 for non-commercial research. Commercial use requires a paid subscription through DrugBank Online. The database provides both open and commercial access tiers with different data availability.

---

## Permissions

What you CAN do with this data (Academic License):

- Use for academic and non-commercial research
- Download bulk data with free registration
- Integrate into non-commercial research tools
- Publish derived analyses with proper citation

---

## Restrictions

What you CANNOT do without additional permission:

- Commercial redistribution of the data
- Incorporate into commercial products or services
- Use for commercial drug discovery without license
- Redistribute raw data for commercial purposes

---

## Special Terms

### Academic Access

Academic users must register with an institutional email address. The CC BY-NC 4.0 license applies to academic downloads.

### Commercial Licensing

Commercial entities must obtain a DrugBank subscription. Contact DrugBank for pricing and terms at https://go.drugbank.com/

## Attribution Template

When attribution is required, use this format:

```
DrugBank data from DrugBank Online (https://go.drugbank.com/).
License: CC BY-NC 4.0.
```

### Citation

```
Wishart DS, et al. (2018) "DrugBank 5.0: a major update to the DrugBank
database for 2018." Nucleic Acids Res. 46(D1):D1074-D1082.
```
