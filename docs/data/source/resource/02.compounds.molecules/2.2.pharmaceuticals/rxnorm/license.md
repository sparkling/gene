# RxNorm License Terms

License details for RxNorm normalized drug nomenclature data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | UMLS License (free) |
| SPDX Identifier | Custom |
| Commercial Use | Yes |
| Attribution Required | Yes |
| Redistribution | Yes (with UMLS compliance) |

---

## Summary

RxNorm is produced by the National Library of Medicine (NLM) and is available through the UMLS (Unified Medical Language System). Access requires free UMLS registration. The license permits commercial use with attribution, but users must comply with UMLS terms of service.

---

## Permissions

What you CAN do with this data:

- Use for any purpose including commercial applications
- Integrate into healthcare IT systems and EHRs
- Build clinical decision support tools
- Create drug interaction databases
- Redistribute with UMLS compliance

---

## Restrictions

What you CANNOT do without compliance:

- Access without UMLS registration
- Redistribute without UMLS license terms
- Remove NLM attribution from derived works

---

## Special Terms

### UMLS Registration

Access to RxNorm requires free UMLS Terminology Services account registration at https://uts.nlm.nih.gov/

### API Access

RxNav API is freely available without authentication for limited queries.

## Attribution Template

When attribution is required, use this format:

```
RxNorm from the National Library of Medicine (https://www.nlm.nih.gov/research/umls/rxnorm/).
```

### Citation

```
Nelson SJ, et al. (2011) "Normalized names for clinical drugs: RxNorm at 6 years."
J Am Med Inform Assoc. 18(4):441-448.
```
