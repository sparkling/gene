# LOTUS License Terms

License details for LOTUS (Linked Open Total Unified Structure-organism) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | CC0 1.0 (Public Domain) |
| SPDX Identifier | CC0-1.0 |
| Commercial Use | Yes (unrestricted) |
| Attribution Required | No (but appreciated) |
| Redistribution | Yes (unrestricted) |

---

## Summary

LOTUS is released under CC0 (Public Domain Dedication), meaning the data is freely available for any use without restrictions. Users may copy, modify, distribute, and use the data for any purpose, including commercial applications, without asking permission. Attribution is appreciated but not legally required.

---

## Permissions

What you CAN do with this data:

- Use for any purpose without restriction
- Commercial use without license fees
- Redistribute in any form
- Create derivative works
- Integrate into proprietary systems

---

## Restrictions

There are no legal restrictions on use of CC0-licensed data.
