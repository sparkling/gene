## Summary

ClassyFire is a freely accessible web-based tool for automated chemical classification. The ClassyFire web service and API are free to use. The ChemOnt chemical ontology can be freely downloaded. Commercial use of the tool in production systems may require contacting the Wishart Lab for clarification.

---

## Permissions

What you CAN do with this data:

- Use the web service for any classification purposes
- Download the ChemOnt ontology
- Use API for programmatic access
- Publish classification results with citation

---

## Restrictions

What you CANNOT do without additional permission:

- Mirror or redistribute the web service
- Large-scale commercial batch processing may require arrangement
- Redistribution of the classification engine

---

## Special Terms

### API Access

A freely accessible API allows access to ClassyFire classifications. Pre-computed classifications for millions of compounds are available.

### Ontology Download

The underlying ChemOnt chemical ontology can be freely downloaded for use in applications.
