## Summary

STITCH data is freely available under CC BY 4.0, the same license as its sister database STRING. Commercial and academic use is permitted with proper attribution. STITCH provides chemical-protein interaction networks using evidence integration methods.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Access via REST API for applications
- Download bulk interaction data
- Integrate into drug discovery pipelines
- Create derivative works and analyses
- Combine with STRING protein networks

---

## Restrictions

What you CANNOT do without additional permission:

- Remove or obscure attribution notices
- Claim STITCH data as original work
- Use STITCH branding without permission

---

## Special Terms

### STRING Integration

STITCH uses the same scoring framework and identifier system as STRING, enabling seamless integration of chemical-protein and protein-protein networks.

### Data Sources

STITCH integrates data from ChEMBL, DrugBank, KEGG, and other sources. Original source attributions are maintained in the data.

## Attribution Template

When attribution is required, use this format:

```
Chemical-protein interaction data from STITCH (http://stitch.embl.de).
License: CC BY 4.0.
```
