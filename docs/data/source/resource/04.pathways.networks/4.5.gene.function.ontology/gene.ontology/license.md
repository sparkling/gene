# Gene Ontology License Terms

License details for Gene Ontology (GO) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Creative Commons Attribution 4.0 International |
| SPDX Identifier | CC-BY-4.0 |
| Commercial Use | Yes |
| Attribution Required | Yes |
| Redistribution | Yes (with attribution) |

---

## Summary

The Gene Ontology (ontology and annotations) is freely available under CC BY 4.0. This permits unrestricted use including commercial applications with proper attribution. GO is the most widely used biological ontology and follows FAIR data principles.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Integrate into commercial software and services
- Redistribute with attribution
- Create derivative works and extensions
- Use for enrichment analysis tools
- Train machine learning models

---

## Restrictions

What you CANNOT do without additional permission:

- Remove or obscure attribution notices
- Claim GO terms as original creations
- Misrepresent the source of annotations

---

## Special Terms

### Ontology vs Annotations

Both the GO ontology structure and the gene-to-GO annotations are covered by CC BY 4.0.

### Contributing Organizations

GO annotations are contributed by many organizations (UniProt, Ensembl, organism databases). The CC BY 4.0 license applies to all consortium-released data.

---

## Attribution Template

When attribution is required, use this format:

```
Gene Ontology data from the GO Consortium (https://geneontology.org).
License: CC BY 4.0.
```
