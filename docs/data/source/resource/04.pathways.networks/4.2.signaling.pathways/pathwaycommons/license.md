# Pathway Commons License Terms

License details for Pathway Commons data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Varies by source (aggregated database) |
| SPDX Identifier | Multiple |
| Commercial Use | Conditional (depends on source) |
| Attribution Required | Yes (cite all sources) |
| Redistribution | Conditional |

---

## Summary

Pathway Commons aggregates data from 22+ pathway and interaction databases. Each source retains its original license terms. Users must check and comply with individual source licenses for their specific use case. Most sources permit academic use; commercial use varies by source.

---

## Permissions

What you CAN do with this data:

- Use for academic research (all sources)
- Access unified BioPAX and SIF formats
- Query via web interface and API
- Download bulk data files
- Use many sources commercially (check individual licenses)

---

## Restrictions

What you CANNOT do without additional permission:

- Ignore source-specific license requirements
- Redistribute without source attribution
- Commercial use of restricted sources (e.g., KEGG)
- Claim aggregated data as original work

## Attribution Template

When using Pathway Commons data, cite the aggregator and original sources:

```
Data from Pathway Commons (https://www.pathwaycommons.org).
Original sources: [List applicable databases].
```

### Citation
