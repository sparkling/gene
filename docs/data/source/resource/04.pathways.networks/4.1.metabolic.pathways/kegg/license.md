# KEGG License Terms

License details for KEGG (Kyoto Encyclopedia of Genes and Genomes) data usage.

---

## Summary

KEGG is not a public domain or freely-licensed database. Academic users have limited free access to the web interface and REST API. Commercial use and bulk FTP access require paid licensing through Pathway Solutions (Kanehisa Laboratories).

---

## Permissions

What you CAN do with this data:

- Use the KEGG website (kegg.jp) for academic research
- Access limited REST API queries for academic purposes
- Cite KEGG data in academic publications
- Use KEGG pathway IDs for cross-referencing

---

## Restrictions

What you CANNOT do without additional permission:

- Use for commercial purposes without a paid license
- Download bulk FTP data without subscription
- Redistribute KEGG data or pathway images
- Provide KEGG-based services without service provider license
- Use analysis tools on Kyoto University site commercially

---

## Special Terms

### Academic Subscription

Academic users requiring FTP access need an academic subscription:
- **End User**: Individual access for a laboratory/department
- **Service Provider**: For developing tools/services using KEGG

### Commercial Licensing

Commercial entities must obtain licenses from Pathway Solutions:
- **End User License**: FTP and website access
- **Business License**: FTP access for integration
