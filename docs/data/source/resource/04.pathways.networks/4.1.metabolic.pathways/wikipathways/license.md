## Summary

WikiPathways is dedicated to the public domain under CC0 1.0, meaning there are no copyright restrictions on use. Users can freely copy, modify, distribute, and perform the work for any purpose, including commercial applications, without asking permission.

## Special Terms

### Courtesy Attribution

While not legally required, citing WikiPathways is appreciated and supports the community-driven curation model:

- Helps track database impact
- Supports funding applications
- Acknowledges curator contributions

### Community Contributions

WikiPathways operates as a wiki; contributions become part of the public domain resource.

---

## Attribution Template

When providing courtesy attribution, use this format:

```
Pathway data from WikiPathways (https://www.wikipathways.org).
License: CC0 1.0 Public Domain.
```
