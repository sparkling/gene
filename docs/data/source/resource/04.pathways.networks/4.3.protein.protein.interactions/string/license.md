# STRING License Terms

License details for STRING (Search Tool for Retrieval of Interacting Genes/Proteins) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|

| Commercial Use | Yes |

| Redistribution | Yes (with attribution) |

---

## Summary

STRING data is freely available under CC BY 4.0, permitting commercial and academic use with attribution. The database is maintained by EMBL and provides comprehensive protein-protein functional associations with a scoring framework.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Access via REST API for applications
- Download bulk network data
- Integrate into commercial products
- Create derivative works and visualizations
- Use for machine learning applications

---

## Restrictions

What you CANNOT do without additional permission:

- Remove or obscure attribution notices
- Claim STRING data as original work
- Use STRING branding without permission

---

## Special Terms

### API Usage

The STRING API is freely accessible. High-volume users should implement appropriate delays and consider local data copies for intensive applications.

### Data Versions

STRING releases annual major versions. Users should cite the specific version used in publications.

---

## Attribution Template

When attribution is required, use this format:

```
Protein association data from STRING (https://string-db.org).
License: CC BY 4.0.
```
