License details for IntAct Molecular Interaction Database data usage.

| Property | Value |

---

## Summary

IntAct data is freely available under CC BY 4.0, allowing commercial and academic use with attribution. IntAct software tools are released under Apache 2.0. As an EMBL-EBI resource and IMEx consortium member, IntAct follows open data principles.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Integrate into commercial products and services
- Redistribute data with proper attribution
- Create derivative works and adaptations
- Access via PSICQUIC API without restrictions
- Download bulk data from FTP

---

## Restrictions

What you CANNOT do without additional permission:

- Remove or obscure attribution notices
- Claim IntAct data as your own creation
- Misrepresent the source of interaction data

---

## Special Terms

### Software License

IntAct software tools (search interface, data pipeline) are licensed under Apache 2.0, permitting commercial use and modification.

### IMEx Consortium

IntAct participates in IMEx data sharing. IMEx-curated interactions are available across consortium member databases with consistent annotation standards.

## Attribution Template

When attribution is required, use this format:

```
Molecular interaction data from IntAct (https://www.ebi.ac.uk/intact/).
European Bioinformatics Institute (EMBL-EBI).
License: CC BY 4.0.
```

### Citation

```
Del Toro N, Shrivastava A, et al. The IntAct database: efficient access to fine-grained molecular interaction data. Nucleic Acids Res. 2022;50(D1):D483-D493.
```
