# JASPAR License Terms

License details for JASPAR (Transcription Factor Binding Profiles) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Creative Commons Attribution 4.0 International |
| SPDX Identifier | CC-BY-4.0 |
| Commercial Use | Yes |
| Attribution Required | Yes |
| Redistribution | Yes (with attribution) |

---

## Summary

JASPAR is completely open-access under CC BY 4.0, permitting unrestricted use including commercial applications. As the largest open database of transcription factor binding profiles, JASPAR is the primary choice for regulatory network analysis in both academic and commercial settings.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Integrate binding motifs into commercial tools
- Redistribute with attribution
- Create derivative databases and analyses
- Use for regulatory element prediction services
- Train machine learning models on motif data

---

## Restrictions

What you CANNOT do without additional permission:

- Remove or obscure attribution notices
- Claim JASPAR profiles as original work
- Misrepresent the source of binding motifs

---

## Special Terms

### Profile Updates

JASPAR releases annual updates (JASPAR2024, etc.). Users should cite the specific version used.

### Motif Formats

All matrix formats (PFM, PWM, MEME, TRANSFAC) are covered by the same CC BY 4.0 license.

## Attribution Template

When attribution is required, use this format:

```
Transcription factor binding profiles from JASPAR (https://jaspar.genereg.net).
License: CC BY 4.0.
```
