# Roadmap Epigenomics License Terms

License details for NIH Roadmap Epigenomics Project data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Open Access (NIH Public Data) |
| SPDX Identifier | Custom |
| Commercial Use | Yes |
| Attribution Required | Yes (cite consortium) |
| Redistribution | Yes |

---

## Summary

NIH Roadmap Epigenomics data is publicly available without restrictions for downloading, analysis, and publication. As an NIH-funded public resource, users may freely use the data for any purpose including commercial applications. Attribution through citation is required.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Download and redistribute data files
- Create derivative works and analyses
- Integrate into commercial products
- Publish results without special permission
- Use for machine learning applications

---

## Restrictions

What you CANNOT do without additional permission:

- Claim data as original work
- Fail to cite the consortium in publications

---

## Special Terms

### Data Availability

Data remains accessible through multiple platforms even though program funding ended in 2017:
- NCBI GEO
- ENCODE portal
- WashU Epigenome Browser
- AWS Open Data Registry

### Protected Data

Some protected/controlled-access data (dbGaP) may require additional approval for access.

---

## Attribution Template

When attribution is required, use this format:

```
Epigenomic data from NIH Roadmap Epigenomics Mapping Consortium (http://www.roadmapepigenomics.org).
```

### Citation

```
Roadmap Epigenomics Consortium. Integrative analysis of 111 reference human epigenomes. Nature. 2015;518(7539):317-330.
```
