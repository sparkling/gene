# ETCM License Terms

License details for ETCM (Encyclopedia of Traditional Chinese Medicine) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Academic Use |
| SPDX Identifier | Custom |
| Commercial Use | Conditional (contact required) |
| Attribution Required | Yes |
| Redistribution | Limited |

---

## Summary

ETCM is available for academic research use. Commercial use requires contacting the database maintainers for licensing arrangements. The database emphasizes traditional TCM theory integration with modern pharmacology.

---

## Permissions

What you CAN do with this data:

- Use for academic research and education
- Publish research findings with citation
- Download data for non-commercial analysis
- Reference in academic publications

---

## Restrictions

What you CANNOT do without additional permission:

- Commercial use or integration
- Bulk redistribution
- Use in commercial products or services
- Commercial data mining

---

## Special Terms

### Academic Access

Academic users can freely browse and search the database through the web interface. Bulk downloads may require registration.

### Commercial Inquiries

For commercial licensing, contact the database maintainers through the website.

---

## Attribution Template

When attribution is required, use this format:

```
TCM data from ETCM - Encyclopedia of Traditional Chinese Medicine (http://www.tcmip.cn/ETCM/).
```

### Citation

```
Xu HY, Zhang YQ, et al. ETCM: an encyclopaedia of traditional Chinese medicine. Nucleic Acids Res. 2019;47(D1):D976-D982.
```
