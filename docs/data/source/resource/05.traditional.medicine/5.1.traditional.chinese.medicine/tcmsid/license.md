# TCMSID License Terms

License details for TCMSID (TCM Systematic Pharmacology Database) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Academic Use Free |
| SPDX Identifier | Custom |
| Commercial Use | Conditional (contact required) |
| Attribution Required | Yes |
| Redistribution | Limited |

---

## Summary

TCMSID is freely available for academic research on TCM pharmacokinetics and systems pharmacology. Commercial use requires contacting the database maintainers. The database provides comprehensive ADME/T properties for TCM compounds.

---

## Permissions

What you CAN do with this data:

- Use for academic systems pharmacology research
- Download ADME property data
- Apply OB/DL filtering for compound prioritization
- Publish research with citation
- Network analysis for academic purposes

---

## Restrictions

What you CANNOT do without additional permission:

- Commercial use or drug development
- Commercial ADMET prediction services
- Integration into commercial platforms
- Commercial redistribution

---

## Special Terms

### ADME Predictions

Oral bioavailability (OB) and drug-likeness (DL) values are computational predictions. Experimental validation is recommended before development decisions.

### Filtering Thresholds

Standard filtering thresholds (OB >= 30%, DL >= 0.18) are suggestions; adjust based on research needs.
