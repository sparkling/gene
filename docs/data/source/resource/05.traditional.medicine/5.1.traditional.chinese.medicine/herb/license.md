# HERB License Terms

License details for HERB (High-throughput Experiment and Reference Database) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Creative Commons Attribution-NonCommercial 4.0 |
| SPDX Identifier | CC-BY-NC-4.0 |
| Commercial Use | No (contact required) |
| Attribution Required | Yes |
| Redistribution | Yes (non-commercial only) |

---

## Summary

HERB is distributed under CC BY-NC 4.0, permitting non-commercial use with attribution. The database uniquely integrates high-throughput gene expression data with TCM information. Commercial use requires contacting the publisher.

---

## Permissions

What you CAN do with this data:

- Use for academic research
- Access gene expression profiles for TCM herbs
- Non-commercial redistribution with attribution
- Create derivative works for research
- Publish academic analyses

---

## Restrictions

What you CANNOT do without additional permission:

- Commercial use of any kind
- Integration into commercial products
- Commercial data services
- For-profit applications

---

## Special Terms

### Expression Data

Gene expression signatures are derived from public GEO experiments. Users should also cite original GEO accessions when applicable.

### Commercial Licensing

For commercial re-use, contact Oxford University Press (reprints@oup.com) for licensing arrangements.

## Attribution Template

When attribution is required, use this format:

```
TCM gene expression data from HERB (http://herb.ac.cn/).
License: CC BY-NC 4.0.
```

### Citation

```
Fang S, Dong L, et al. HERB: a high-throughput experiment- and reference-guided database of traditional Chinese medicine. Nucleic Acids Res. 2021;49(D1):D1197-D1206.
```
