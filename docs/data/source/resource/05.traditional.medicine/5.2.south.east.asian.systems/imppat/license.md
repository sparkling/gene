# IMPPAT License Terms

License details for IMPPAT 2.0 (Indian Medicinal Plants, Phytochemistry And Therapeutics) data usage.

---

## Quick Reference

| Property | Value |
|----------|-------|
| License | Creative Commons Attribution 4.0 / ND for derivatives |
| SPDX Identifier | CC-BY-4.0 (with restrictions) |
| Commercial Use | Yes (with attribution) |
| Attribution Required | Yes |
| Redistribution | Conditional |

---

## Summary

IMPPAT 1.0 was released under CC BY 4.0. IMPPAT 2.0 includes additional restrictions on derivative works (No Derivatives for sharing). Academic and commercial use is permitted with proper attribution, but derivative work sharing may be restricted.

---

## Permissions

What you CAN do with this data:

- Use for academic and commercial research
- Download compound structures and properties
- Access 1,875 molecular descriptors per compound
- Use ADMET predictions and drug-likeness scores
- Publish research with citation

---

## Restrictions

What you CANNOT do without additional permission:

- Share derivative works publicly (IMPPAT 2.0)
- Redistribute modified databases
- Remove attribution notices

---

## Special Terms

### Structure Downloads

Compound structures are available in multiple formats (SDF, MOL, MOL2, PDB, PDBQT) for computational applications.

### Target Predictions

Target predictions are from STITCH with score >= 700. Lower-confidence interactions are excluded.

### GitHub Resources

Analysis scripts and supplementary data are available on GitHub (https://github.com/asamallab/IMPPAT2).
