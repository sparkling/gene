# NAPRALERT Download Instructions

## Quick Start

```bash
# Subscription-based access required
# URL: https://napralert.pharmacy.uic.edu/
```

## Prerequisites

- Institutional or commercial subscription
- Web browser for query interface
- Contact UIC for licensing

## Access Model

NAPRALERT uses a subscription-based access model:

| Tier | Access Level | Details |
|------|--------------|---------|
| Free | Limited queries | Basic searches |
| Academic | Institutional unlimited | University subscription |
| Commercial | Full access + reports | Enterprise license |

## Subscription Process

1. Visit https://napralert.pharmacy.uic.edu/
2. Review subscription options
3. Contact UIC licensing office
4. Complete subscription agreement
5. Receive access credentials

## Query Interface

After subscription:

```
Web Interface Features:
- Organism search (by species, family, genus)
- Compound search (by name, class)
- Activity search (by pharmacological effect)
- Literature search (by author, journal, year)
- Combined searches with Boolean operators
```

## Data Fields

### Organism Record
- Scientific name
- Family, genus, species
- Common names
- Geographic distribution
- Ethnobotanical uses

### Compound Record
- Compound name
- CAS number
- Molecular formula
- Structure (when available)
- Biological source
- Literature references

### Activity Record
- Activity type
- Test system
- Dose/concentration
- Effect observed
- Literature citation
